Privacy Policy

Last updated: 2026-05-12  ·  Version 2.0.0

Overview

This Privacy Policy explains how ezXSave ("the Extension") handles user data. The Extension is a lightweight Chrome extension that adds a convenient download button to X (Twitter) videos and GIFs, allowing users to save media in its original MP4 quality.

The Extension operates entirely within your browser. No user data is ever transmitted to any external server, analytics service, or third party.

1. Data We Process

The Extension temporarily processes public information rendered on the active X page, specifically:

  • Tweet IDs — extracted from the page URL or tweet <article> elements to identify the video or GIF to download.

This data is used solely to request the video source URL directly from Twitter's own Syndication API.

2. Comprehensive Disclosure

2-a. How We Collect, Use, and Store

  • Collect: When you click the download button, the Extension reads the Tweet ID and securely fetches video metadata from Twitter's official Syndication API.
  • Use: The fetched metadata is used exclusively to find the highest-bitrate MP4 URL and trigger a local file download.
  • Store: The Extension does not store any persistent data, user profiles, or tracking history.
  • Share: Data is never shared with third parties. All API requests are made directly between your browser and Twitter's servers.

2-b. Parties With Whom Data Is Shared

We do not share any user data with third parties. The Extension contains no tracking analytics, advertising SDKs, remote servers, or cloud-sync functionality.

3. Permissions Used

  • downloads — Required to save the extracted MP4 files directly to your device's local storage.
  • host_permissions: https://x.com/*, https://twitter.com/*, https://cdn.syndication.twimg.com/*, https://video.twimg.com/* — Required to detect videos on X, inject the download button into the page UI, and securely retrieve media metadata directly from Twitter's servers.

Additional Information

  • Security: All data processing is strictly confined to your local browser environment.
  • Changes: If future feature updates require changes to this policy, the updated date will be reflected at the top of this page.
